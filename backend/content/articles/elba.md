1. **Год бесплатно для новых ИП** — Тариф «Премиум» на год при регистрации. ИП должно быть моложе 3 месяцев.
2. **Бесплатно через банк-партнёр** — Тинькофф, Точка, Альфа дают Эльбу бесплатно в первый год. Проверьте «Подарки» в ЛК банка.
3. **Интеграция с банком** — Подключите выписку по API — операции загружаются автоматически.
4. **Автоматические декларации** — Эльба формирует декларацию УСН, КУДиР и платёжки по взносам.
5. **На патенте — минимальный тариф** — Отчётности почти нет. Если умеете работать с nalog.gov.ru — оставайтесь на бесплатном.
